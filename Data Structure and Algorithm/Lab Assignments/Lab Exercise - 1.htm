Lab 01 – Linked Lists

In this lab you will work with linked lists. No submission is required for this lab, but you must get your work checked by a lab TA.   

You can use the linked list class from a book by Robert Lafore as posted here (Link, LinkList). It provides the following  methods:  

insertFirst, find, delete, displayList, insertLast, clear, element, get, removeFirst

In the following, assume list1 is the LinkList object upon which the method is invoked. The first node in the list is assumed to be at index 0. 

1.1.   Write and test this method: 

int size()

// returns: the number of nodes in list1; 

For example, if list1 is {33, 55, 77, 99}, then  size() will return 4. 

1.2.   Write and test this method: 

int sum()

// returns: the sum of the integers in list1; 

For example, if  list1 is {25, 45, 65, 85}, then sum() will return 220. 

1.3.   Write and test this method: 

void removeLast()

// precondition: list1 has at least two nodes;

// postcondition: the last node in list1 has been deleted; 

For example, if list1 is {22, 44, 66, 88}, then removeLast() will change it to {22, 44, 66}. 

1.4.   Write and test this method: 

 LinkList copy()

 // returns: a new list that is a duplicate of list1; 

     Note that the new list must be completely independent of list1. Changing one list should have no effect upon the other. 

1.1.   Write and test this method:

LinkList sublist(int p, int q)

// returns: a new list that contains copies of the q-p nodes of list1, starting with

// node number p (starting with 0); 

For example, if list1 is {22, 33, 44, 55, 66, 77, 88, 99}, then sublist(2, 7) will return the new list {44, 55, 66, 77, 88}. Note that the two lists must be completely independent of each other. Changing one list should have no effect upon the other. 

1.2.   Write and test this method: 

LinkList append(LinkList list2)

// precondition: list1 has at least one node;

// postconditions: list1 has list2 appended to it; 

For example, if list1 is {22, 33, 44, 55} and list2 is {66, 77, 88, 99}, then append(list2) will change list1 to {22, 33, 44, 55, 66, 77, 88, 99}. Note that no new list is created by this method.  

1.3.   Write and test this method: 

LinkList concat(LinkList list2)

// returns: a new list that contains a copy of list1,

// followed by a copy of list2;  

For example, if list1 is {22, 33, 44, 55} and list2 is {66, 77, 88, 99}, then concat(list2) will return the new list {22, 33, 44, 55, 66, 77, 88, 99}. Note that the three lists should be completely independent of each other. Changing one list should have no effect upon the others. 

1.4.   Write and test this method: 

void replace(int i, int x)

// replaces the ith element with x;  

For example, if list1 is {22, 33, 44, 55, 66, 77, 88, 99}, then replace(2, 50) will change list1 to {22, 33, 50,  55,  66, 77, 88, 99}.  

1.5.   Write and test this method: 

void swap(int i, int j)

// swaps the ith element with the jth element;  

For example, if list1 is {22, 33, 44, 55, 66, 77, 88, 99}, then swap(2, 5) will change list1 to {22, 33, 77,  55,  66, 44, 88, 99}.  

1.6.   Write and test this method: 

void sort()

// postconditions: list1 is sorted in ascending order of keys;  

For example, if list1 is {33, 99, 88,  44, 55, 77, 66, 22}, then sort() will change list1 to {22, 33,  44, 55,  66, 77, 88, 99}.  

1.7.   Write and test this method: 

void merged(LinkList list2)

// precondition: list1 and list2 are both in ascending order;

// returns: a new list that contains all the elements of list1 and list2 in ascending order 

For example, if list1 is {22, 33, 55, 88} and list2 is {44, 66, 77, 99}, then merged(list2) will return the new list {22, 33, 77,  55,  66, 44, 88, 99}. Note that the three lists should be completely independent of each other. Changing one list should have no effect upon the others.
